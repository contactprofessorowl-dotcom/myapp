# Application Overview

The application is a quiz app that allows users to test their knowledge on various topics. The app has a home screen, a quiz screen, a results screen, a settings screen, and an account screen. The app also has a theme provider that allows users to switch between light and dark mode.

# Project Outline

* **`lib/main.dart`**: The main entry point of the application. It initializes the app, sets up the providers, and configures the router.
* **`lib/home_screen.dart`**: The home screen of the application. It displays a welcome message and provides options to start the quiz or view the user's profile.
* **`lib/quiz_screen.dart`**: The quiz screen. It displays the quiz questions and allows the user to select their answers.
* **`lib/results_screen.dart`**: The results screen. It displays the user's score and provides an option to retake the quiz.
* **`lib/settings_screen.dart`**: The settings screen. It allows the user to configure the app's settings, such as the theme.
* **`lib/account_screen.dart`**: The account screen. It displays the user's profile information.
* **`lib/theme.dart`**: The theme data for the application. It defines the color scheme, typography, and component styles for the light and dark themes.
* **`lib/providers.dart`**: The providers for the application. It includes the `ThemeProvider` and `UserData` providers.
* **`pubspec.yaml`**: The project's configuration file. It lists the project's dependencies and other settings.
